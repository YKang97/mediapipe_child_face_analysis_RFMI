# Demo Outputs

This folder contains static demonstration outputs generated from the synthetic open-eye example image.

## Figures

- `figures/demo_full_face_overlay.jpg`: full-face MediaPipe landmark overlay.
- `figures/demo_eye_zoom_overlay.jpg`: eye-region landmark overlay.
- `figures/demo_rfmi_lines_overlay.jpg`: RFMI distance-line overlay.

## Tables

- `tables/demo_rfmi_subject_indices.csv`: one-row subject-level RFMI output from the synthetic example.
- `tables/demo_rfmi_summary.csv`: descriptive RFMI summary table from the synthetic example.

These files are included only to show the expected output format. They are not study data.
