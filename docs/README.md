# Demo Outputs

This folder contains static demonstration outputs generated from the synthetic open-eye example image. These files are included so that readers can preview the expected notebook outputs directly from the repository README.

## Figures

- `figures/rfmi_six_part_overview.png`: six-part workflow overview used on the repository front page.
- `figures/demo_synthetic_input.jpg`: clean synthetic open-eye input image.
- `figures/demo_full_face_overlay.jpg`: full-face MediaPipe landmark overlay.
- `figures/demo_eye_zoom_overlay.jpg`: eye-region landmark overlay.
- `figures/demo_rfmi_lines_overlay.jpg`: RFMI distance-line overlay.
- `figures/demo_rfmi_subject_table.png`: subject-level RFMI table preview.
- `figures/demo_rfmi_summary_table.png`: descriptive RFMI summary table preview.

## Tables

- `tables/demo_rfmi_subject_indices.csv`: one-row subject-level RFMI output from the synthetic example.
- `tables/demo_rfmi_summary.csv`: descriptive RFMI summary table from the synthetic example.

These files are demonstration outputs only. They are not study data.
